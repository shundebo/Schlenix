http.server# Schlenix

Schlenix 是 Schlen 联邦共和国的官方操作系统 - 一个基于 Web 技术构建的前端伪操作系统，采用类 LXQt 风格的桌面环境设计。

## 🎨 特性

- **完整的桌面环境**：类 LXQt 风格的现代化界面
- **窗口管理系统**：支持拖动、调整大小、最小化、最大化
- **任务栏**：应用菜单、窗口列表、系统托盘
- **多个内置应用**：
  - 📁 文件管理器 - 浏览虚拟文件系统
  - 💻 终端 - 执行系统命令
  - 🌐 浏览器 - 内嵌网页浏览
  - 📝 文本编辑器 - 编辑文本文件
  - 🔢 计算器 - 基础计算功能
  - ⚙️ 系统设置 - 自定义系统外观
  - ℹ️ 关于 - 系统信息

## 🚀 快速开始

直接在浏览器中打开 `index.html` 文件即可。

或使用本地服务器：
```bash
python -m http.server 8000
```

然后访问 `http://localhost:8000`

## ⌨️ 快捷键

- `Ctrl+Alt+T` - 打开终端
- `Ctrl+Alt+F` - 打开文件管理器
- `Ctrl+Alt+B` - 打开浏览器

## 🎮 使用说明

### 桌面操作
- **单击图标**：选择图标
- **双击图标**：打开应用
- **右键桌面**：显示上下文菜单

### 窗口操作
- **拖动标题栏**：移动窗口
- **双击标题栏**：最大化/还原窗口
- **拖动边缘**：调整窗口大小

## 🛠️ 技术栈

- 纯 HTML5 + CSS3 + JavaScript
- 无外部依赖
- 模块化架构

## 📝 终端命令

- `help` - 显示帮助信息
- `clear` - 清空终端
- `neofetch` - 显示系统信息
- `about` - 关于 Schlenix
