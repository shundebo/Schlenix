http.server# Schlenix

Schlenix 是 Schlen 联邦共和国的官方操作系统 - 一个基于 Web 技术构建的前端伪操作系统，采用类 LXQt 风格的桌面环境设计。

## 🎨 特性

- **完整的桌面环境**：类 LXQt 风格的现代化界面
- **窗口管理系统**：支持拖动、调整大小、最小化、最大化、右键菜单
- **任务栏**：应用菜单、窗口列表、系统托盘
- **持久化存储**：使用 LocalStorage 保存文件和设置
- **通知系统**：优雅的桌面通知提示
- **虚拟文件系统**：完整的文件操作支持（创建、删除、重命名、移动）
- **22 个内置应用**：
  - 📁 文件管理器 - 完整的文件系统操作
  - 💻 终端 - 支持文件操作命令和 Tab 补全
  - 🌐 浏览器 - 多标签页、下载管理、Bing 搜索
  - 📝 文本编辑器 - 编辑并保存文件到虚拟文件系统
  - 🔢 计算器 - 基础四则运算
  - 📋 记事本 - 快速笔记管理
  - 🖼️ 图片查看器 - 支持本地图片上传
  - 📊 任务管理器 - 系统资源监控
  - 🕐 时钟 - 时钟、秒表、计时器
  - 🎵 音乐播放器 - 支持本地音频文件
  - 📈 系统监控 - 实时 CPU/内存图表
  - ⌨️ 快捷方式管理器 - 自定义快捷键
  - 📝 Markdown 预览器 - 实时预览和导出
  - 🍅 番茄钟 - 工作计时和休息提醒
  - 🎨 主题编辑器 - 自定义系统主题
  - 🖌️ 画板 - 绘图工具和图片编辑
  - 🐍 贪吃蛇 - 经典贪吃蛇游戏
  - 🎮 俄罗斯方块 - 经典方块游戏
  - 🎯 2048 - 数字合并游戏
  - 💣 扫雷 - 经典扫雷游戏
  - ⚙️ 系统设置 - 自定义外观并持久化保存
  - ℹ️ 关于 - 系统信息



## ⌨️ 快捷键

- `Ctrl+Alt+T` - 打开终端
- `Ctrl+Alt+F` - 打开文件管理器
- `Ctrl+Alt+B` - 打开浏览器
- `Ctrl+Alt+N` - 打开记事本
- `Ctrl+Alt+M` - 打开任务管理器
- `Ctrl+Alt+I` - 打开图片查看器
- `Ctrl+Alt+C` - 打开时钟
- `Ctrl+Alt+P` - 打开音乐播放器
- `Ctrl+Alt+S` - 打开系统监控
- `Ctrl+Alt+K` - 打开快捷方式管理器

## 🎮 使用说明

### 桌面操作
- **单击图标**：选择图标
- **双击图标**：打开应用
- **右键桌面**：显示上下文菜单

### 窗口操作
- **拖动标题栏**：移动窗口
- **双击标题栏**：最大化/还原窗口
- **拖动边缘**：调整窗口大小

## 🛠️ 技术栈

- 纯 HTML5 + CSS3 + JavaScript
- 无外部依赖
- 模块化架构

## 📝 终端命令

### 系统命令
- `help` - 显示帮助信息
- `clear` - 清空终端
- `neofetch` - 显示系统信息
- `about` - 关于 Schlenix
- `uname` - 显示系统信息
- `uptime` - 显示运行时间
- `free` - 显示内存使用
- `df` - 显示磁盘使用
- `ps` - 显示进程列表
- `history` - 显示命令历史

### 文件操作
- `cd` - 切换目录
- `pwd` - 显示当前路径
- `ls` - 列出文件
- `cat` - 显示文件内容
- `mkdir` - 创建目录
- `touch` - 创建文件
- `rm` - 删除文件/目录 (使用 -r 递归删除)
